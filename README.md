# vue-video-hls

## Project setup 安装项目
```
yarn install
```

### Compiles and hot-reloads for development 运行开发环境
```
yarn serve
```

### Compiles and minifies for production 生产环境构建
```
yarn build
```

### Lints and fixes files 代码格式检查修复
```
yarn lint
```
#**特性介绍**
* 支持m3u8 rtmp hls等
* 播放速度可选
* 支持循环播放
* 支持全屏播放
* 支持音频设置
* 兼容HTML5
* 部署简单，直接运行脚本启动
